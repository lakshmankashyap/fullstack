# flex-basis
制定某一个`item`在主轴上的大小，或者在主轴上相对于`flex-container`大小的比例关系。

### Value

```js
.flex-item {
  -webkit-flex-basis: auto | <width>; /* Safari */
  flex-basis:         auto | <width>;
}
```
