# align-items
当`flex items`在主轴上只有`一排`时，`align-items`属性主要用于设置`交叉轴`上`flex items`的排列方式。

## stretch
### Value

```JavaScript
.flex-container {
  -webkit-align-items: stretch; /* Safari */
  align-items: stretch;
}
```
