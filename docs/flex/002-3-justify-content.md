# justify-content
`justify-content`主要用设置`flex items`在`容器`里面严着主轴的排列方式。

## flex-start
### Value

```JavaScript
.flex-container {
  -webkit-justify-content: flex-start; /* Safari */
  justify-content: flex-start;
}
```
