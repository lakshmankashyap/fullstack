# flex-direction
flex-direction主要用在容器中设置容器的`主轴`，flex-direction可以设置两个方向，分别为`row`和`column`

## row

### Value

```JavaScript
.flex-container {
  -webkit-flex-direction: row; /* Safari */
  flex-direction: row;
}
```
